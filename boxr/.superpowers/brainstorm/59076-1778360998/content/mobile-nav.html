B — Нижняя навигация (Tab Bar)

Фиксированный таббар внизу экрана — стандарт для мобильных приложений. Удобно для большого пальца. Нужен тот же layout wrapper.

C — Минимально: только inline навигация

Не добавляем единый shell. Каждая страница сохраняет собственную навигацию (стрелки назад, вкладки). Меньше изменений, но навигация между разделами не унифицирована.

💡 Рекомендация: A (шапка + drawer) — больше места на экране для контента, работает одинаково хорошо для всех ролей, легко расширяется. Tab Bar (B) удобнее для частых переходов, но в BOXR разделов мало и они роль-зависимые.

A — Кнопка в каждой вкладке

В шапке каждой вкладки (Жеребьёвка / Расписание / Результаты) есть своя кнопка «Распечатать». Видна только когда есть что печатать.

B — Отдельная вкладка «Печать»

Отдельная вкладка с тремя кнопками — по одной на каждый документ. Всё в одном месте, не нужно переключать вкладки.
